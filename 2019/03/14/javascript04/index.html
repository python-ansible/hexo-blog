7种数据类型

JavaScript 是一种弱类型或者说动态语言。
这意味着你不用提前声明变量的类型，在程序运行过程中，类型会被自动确定。
这也意味着你可以使用同一个变量保存不同类型的数据。
number/string/boolean/symbol/null/undefined/object
