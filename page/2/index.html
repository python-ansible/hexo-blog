css-flex

Posted on 2019-02-28 | In css

flex 之前

normal flow
float + clearfix
position relative + absolute
display inline-block
负 margin

flex 布局特点

  • 与方向无关
  • 空间自动分配、自动对齐
  • 使用于简单线性布局，负责的可以使用grid

flex container 属性

  • flex-direction 方向
    • row
    • row reverse
    • column
    • column reverse
  • flex-wrap 换行
    • wrap
    • nowrap
    • wrap-reverse
  • flex-flow 上面两项的简写
    • row wrap
  • justify-content 主轴方向对齐方式
    • space-between
    • space-around
    • flex-start
    • flex-end
    • center
  • align-items 侧轴对齐方式
    • stretch 伸展
    • flex start
    • flex end
    • center
    • baseline 文字基线对齐
  • align-content 多行/列内容对齐方式
    • stretch 每一行拉伸填满容器
    • flex start 行集中在顶部
    • flex end 行集中在底部
    • space-around 每行周围保持相等距离
    • space-between 行与行之间保持相等距离
    • center

flex item 属性

  • flex-grow 增长比例
    • 定义如何分配多余空间，值为数字。
  • flex-shrink 收缩比例 空间不足时
  • flex-basis 默认原始大小
    • 指定每个item 占位大小，单位px
  • flex 上面三项的缩写
  • order 顺序(代替双飞翼)
    • 指定每个item的顺序
    • order: 1;
  • align-self 指定item自身对齐方式
    • 优先级高于container

闭包

将函数内外连接的桥梁

作用：

  • 避免使用全局变量
  • 读取函数内部变量

    在退出函数之前把不使用的局部变量全部删除
